# Remove-SmaServiceDeskTicket

## SYNOPSIS

## SYNTAX

```
Remove-SmaServiceDeskTicket -Id <Int32> [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Deletes a service desk ticket.

## EXAMPLES

### EXAMPLE 1
```powershell
PS> Remove-SmaServiceDeskTicket -TicketID 1234
```

Deletes a ticket with ID 1234
