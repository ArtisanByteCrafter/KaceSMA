# Get-SmaAssetType

## SYNOPSIS

## SYNTAX

```
Get-SmaAssetType [[-QueryParameters] <String>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Returns information about an SMA asset type.

## EXAMPLES

### EXAMPLE 1
```
Get-SmaAssetType
```

Retrieves information about all asset types within an org.
